# Bot-Tiro-ao-Alvo

## Descrição
Um bot feito para o jogo https://mouseaccuracy.com/, ele é capaz de detectar e clicar nos alvos encontrados na tela à medida que eles vão aparecendo.

## Bibliotecas
- opencv-python
- pyautogui
- keyboard
- pillow
